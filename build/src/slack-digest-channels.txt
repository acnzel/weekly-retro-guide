# slack-digest 기본 채널 리스트
# - 한 줄에 채널 ID 하나. 줄 맨 앞 토큰만 ID로 읽고, '#' 이후는 주석(채널명 메모용).
# - 빈 줄/주석 줄은 무시됨.
# - /slack-digest 에 인자로 채널 ID를 주면 이 파일 대신 그 채널들만 처리됨.
# - 채널 ID 얻는 법: Slack에서 채널 이름 우클릭 → "채널 세부정보 보기" 맨 아래, 또는 채널 링크 끝의 C로 시작하는 값.
#
# 예시(주석을 풀고 본인 채널 ID로 바꾸세요):
# C0123ABCDEF   # 백엔드-알림
# C0456GHIJKL   # 매니저-운영
